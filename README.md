# Emax Client Knowledge Base — Paid Social

An Obsidian + Claude Code knowledge base for managing Emax clients. Built for paid social consultants (Meta Ads primarily).

## What it does

- **Auto-context:** Mention a client and Claude loads their full profile, activity log, and strategy
- **Auto-logging:** After work, Claude logs what was done and why — feeds into CRM comments
- **Meeting processing:** Share a Fireflies link and get structured notes, action items, and follow-up emails
- **Weekly CRM review:** Friday summaries compiled into Data Pulse-ready comments
- **Knowledge compounding:** Solved problems are documented and auto-retrieved next time
- **CRM sync:** Pulls client notes, grades, and contracts from Supabase

## Getting started

1. Clone this repo
2. Install [Obsidian](https://obsidian.md) and open this folder as a vault
3. Open this folder in [Claude Code](https://claude.ai/download)
4. Say anything — the setup wizard starts automatically

The wizard asks your name, role, connects your tools (Supabase CRM, Google Calendar, Claude in Chrome, Fireflies), and sets up your client folders.

## Requirements

- [Claude Code](https://claude.ai/download)
- [Obsidian](https://obsidian.md) (free)
- Emax Supabase CRM credentials (ask the dev team)
- [Claude in Chrome](https://chromewebstore.google.com) extension (for Fireflies processing)

## Optional

- Todoist (task management and Monday scrum generation)
- Google Drive MCP (access shared client documents)
